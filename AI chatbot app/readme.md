# AI Chatbot Using LLMs

## Description
A full-stack AI chatbot application built using FastAPI and Streamlit, integrated with Google Gemini LLM API for intelligent responses.

## Technologies
- Python
- FastAPI
- Streamlit
- Google Gemini API
- REST APIs

## How to Run
1. Set Gemini API key in backend/.env
2. Start backend:
   uvicorn main:app --reload
3. Start frontend:
   streamlit run app.py

## Features
- Real-time AI responses
- Chat-based UI
- Secure API key handling
- Modular architecture
